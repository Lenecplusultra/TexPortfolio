## 🌟 Featured Projects

### ⚙️ Distributed Key-Value Store & Rate Limiter
- **Stack**: Go, TCP Networking, Consistent Hashing, LRU Eviction, Append-Only Log
- Built from scratch in Go with GET, SET, DEL, and TTL expiry over raw TCP — no framework dependencies
- Implemented LRU eviction, append-only log persistence, and primary-replica replication for fault tolerance
- Applied consistent hashing for distributed sharding; built token-bucket rate limiter for concurrent load control

### 🏪 ANBELO Retail Intelligence Platform (ARIP)
- **Stack**: Java, Spring Boot, PostgreSQL, React, Docker, AWS
- Cloud-native analytics platform built on SAGE ERP for multi-store retail operations
- Centralizes sales and inventory data across stores; delivers executive dashboards, inventory alerts, and revenue analytics

### 🤖 MCP-Powered Internal Engineering Assistant (OpsPilot)
- **Stack**: Python, FastAPI, LangChain, OpenAI API, PostgreSQL, React
- AI-powered assistant enabling engineering teams to query metrics, replay failed events, and troubleshoot systems through natural language
- Enforces RBAC with per-role tool invocation policies and full audit logging; integrates GitHub, DB schemas, logs, and runbooks

### 📡 Event-Driven Order Fulfillment Platform
- **Stack**: Python, FastAPI, Kafka, Redis, PostgreSQL, Docker, Kubernetes, OpenTelemetry
- High-throughput multi-service backend using Kafka for async event streaming across full order lifecycle
- Implements Saga orchestration with compensating transactions; deployed to Kubernetes with horizontal autoscaling
- Full distributed tracing with OpenTelemetry dashboards tracking latency percentiles and failure rates

### 🧠 Cogitons — Knowledge Platform
- **Stack**: Next.js, FastAPI, PostgreSQL
- Subject-centered knowledge platform designed for Cameroonian students and young professionals
- Product-first workflow: full requirements documentation, architecture design, user flows, and database schema

### 📊 AI-Powered Internal Doc Assistant
- **Stack**: React, FastAPI, LangChain, GPT-4, PostgreSQL
- ChatGPT-style assistant for semantic search over 200+ internal documents using GPT-4 and vector search
- Real-time response generation with WebSockets; deployed on Vercel + Render
